# Agent-skills-setup-for-AntiGravity

## Tổng quan
Bộ công cụ scaffolding/CLI giúp thiết lập nhanh hệ thống "skills" cho AI agent chạy trong Google AntiGravity IDE. Đây là dự án nổi bật nhất trong portfolio — 506★ trên GitHub, được cộng đồng vibe-coding và AI-agent builder đón nhận.

## Vấn đề giải quyết
Khi làm việc với agent (Claude, Gemini, các LLM agentic khác), việc tạo cấu trúc "skill" chuẩn (metadata, script, hướng dẫn dùng) thường phải làm thủ công, lặp đi lặp lại. Dự án tự động hoá việc scaffold này, giúp developer/prompt engineer dựng agent mới trong vài phút thay vì vài giờ.

## Công nghệ
Python, thiết kế dạng CLI, tập trung vào developer tooling và prompt engineering. Gắn với hệ sinh thái AntiGravity (IDE agentic của Google), ai-agents, cli, scaffolding.

## Điểm nổi bật
- 506 sao GitHub — minh chứng rõ nhất cho năng lực kỹ thuật và khả năng tạo ra công cụ hữu ích cho cộng đồng.
- Có bản deploy trên Vercel (`Agent-skills-setup-for-AntiGravity-vercel`) — mở rộng thành web tool dễ tiếp cận hơn CLI thuần.
- Là "cổng vào" chính khi người xem GitHub tìm hiểu về Khoa — nên đây là project cần thumbnail ấn tượng nhất.

## Vai trò
Tác giả và người duy trì chính (founder-mindset), tự thiết kế kiến trúc CLI và mô hình skill.

## Gợi ý ý tưởng hình ảnh (để tự viết prompt Gemini)
- Ẩn dụ trung tâm: một "bộ não" hoặc "lõi điều phối" AI toả ra nhiều nhánh nhỏ (mỗi nhánh = 1 skill/agent con) — thể hiện tính scaffolding/orchestration.
- Có thể dùng hình ảnh cấu trúc cây thư mục (file tree) phát sáng, biến thành mạng lưới neural.
- Giữ tông màu đã thiết lập cho site: nền đen tuyệt đối, neon cyan #00C8FF, sàn phản chiếu ướt kiểu Tron.
- Tránh dùng chữ/logo thật trong ảnh (thumbnail chỉ minh hoạ ý tưởng, không phải UI thật).
